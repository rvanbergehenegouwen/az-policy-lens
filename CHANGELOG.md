# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-07-21

### Added
- Initial release of Azure Policy Lens
- Multi-tab dashboard (Dashboard, Policies, Import, Users, Settings)
- Compliance framework analysis (Cyber Essentials, MCSB)
- Policy assignment visualization and drilldown
- CSV import/export capabilities
- Role-based access control (admin, superuser, user)
- Audit logging for sensitive operations
- Azure App Service deployment ready
- Application Insights integration
- SQLite database with auto-initialization
- React 19 frontend with Vite and TypeScript
- Express.js backend with comprehensive API
- GitHub Actions CI/CD workflow
- Azure Bicep infrastructure as code

### Features
- Real-time compliance tracking
- Policy search and filtering
- User management interface
- Audit trail viewer (admin only)
- Performance optimized queries
- Responsive design with BAM branding
- Passwordless/Azure AD auth ready

**Initial Release** — Foundation for Azure Policy compliance monitoring and management.
